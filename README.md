# ⚖️ Planalto Reader Pro

Ler a "lei seca" no site do Planalto sempre foi um desafio para os olhos e para a organização dos estudos. O **Planalto Reader Pro** é um script criado de concurseiro para concurseiro para transformar completamente a sua experiência de leitura da legislação brasileira, trazendo ferramentas de um Vade Mecum físico e digital para dentro do site oficial do governo.

## 💖 Gratuito para Sempre: Solidariedade Concurseira

Este script foi desenvolvido com muito carinho para ajudar a comunidade de estudantes que, assim como eu, enfrenta as agruras da "lei seca". Ele é gratuito hoje e **será gratuito para sempre**. É uma pequena contribuição de apoio mútuo para quem está na jornada rumo à aprovação.

---

## 🌟 Funcionalidades: Como usar e Para que servem

### 🌙 1. Dark Mode Nuclear
* **Como funciona:** Aplica um fundo escuro e texto claro de forma absoluta, corrigindo até mesmo as tabelas quebradas e textos sobrepostos de leis muito antigas (como a LINDB e o CTN).
* **Para que serve:** Salvar a sua visão. A leitura prolongada de telas brancas causa fadiga ocular extrema, prejudicando o rendimento em sessões longas de estudo.

### 📖 2. Estrutura "Vade Mecum" (Escadinha Juríca)
* **Como funciona:** O script identifica automaticamente Caputs, Parágrafos, Incisos e Alíneas, aplicando margens, recuos e bordas coloridas laterais. 
* **Para que serve:** Memória fotográfica e espacial. O cérebro memoriza melhor quando o texto tem uma estrutura visual clara, permitindo que você bata o olho e entenda a hierarquia da norma instantaneamente.

### 🖍️ 3. Sistema de Grifos Multicores
* **Como funciona:** Selecione qualquer trecho do texto com o mouse. Um menu flutuante aparecerá permitindo grifar o trecho em 5 cores diferentes (Amarelo, Verde, Azul, Rosa e Laranja). Para apagar, basta clicar no grifo e no botão de exclusão (✕). Os grifos ficam salvos no seu navegador.
* **Para que serve:** Categorização. Você pode usar amarelo para a regra geral, rosa para exceções, verde para prazos e azul para palavras-chave, otimizando as revisões futuras.

### 📝 4. Glosas (Anotações de rodapé)
* **Como funciona:** Ao clicar em um grifo, você pode adicionar uma anotação clicando no botão "✎ glosa". O texto grifado ganhará uma pequena seta (↗) indicando que há uma nota ali. Ao passar o mouse pela tela, suas anotações aparecerão discretamente no canto direito.
* **Para que serve:** Perfeito para anotar súmulas do STF/STJ, referências de repercussão geral, teses repetitivas ou remissões a outros artigos sem poluir o texto principal.

### 🎯 5. Controle de Incidência (Os "Pontinhos")
* **Como funciona:** Ao lado esquerdo de cada "Artigo", você verá um espaço sutil. Clique ali com o botão esquerdo para adicionar um "pontinho" colorido. Clique com o botão direito para remover. A cor e o formato dos pontos mudam conforme a quantidade aumenta.
* **Para que serve:** Mapeamento de calor de incidência em provas. Sempre que você resolver uma questão de concurso e a resposta for a literalidade daquele artigo, você vai lá e adiciona um ponto. Com o tempo, as leis estarão mapeadas: você saberá exatamente quais artigos despencam em prova e quais nunca são cobrados.

### 🚑 6. Painel de Órfãos (Lost & Found)
* **Como funciona:** Se a lei for alterada ou revogada pelo Legislativo, o trecho que você grifou pode sumir ou mudar. O script detecta isso e envia seu grifo (e a glosa) para um painel de alerta no canto inferior direito.
* **Para que serve:** Segurança. Garante que você nunca perca suas anotações preciosas só porque o texto da lei sofreu atualização.

## ⚙️ Como Instalar

Para usar o Planalto Reader Pro, você precisa de um gerenciador de scripts no seu navegador. É muito simples:

1. Instale a extensão **Tampermonkey** no seu navegador (disponível para Chrome, Edge, Firefox, Safari, etc).
2. Após instalar a extensão, **[CLIQUE AQUI PARA INSTALAR O SCRIPT]**(Substitua_este_texto_e_os_colchetes_pelo_seu_link_RAW_do_github).
3. O Tampermonkey abrirá uma nova aba. Clique em **"Instalar"**.
4. Pronto! Acesse qualquer lei no site `planalto.gov.br` e o script já estará funcionando magicamente.
