# planalto-reader-pro
O Planalto Reader Pro é o script definitivo para estudar a lei seca. Ele transforma o site do Planalto com um Dark Mode absoluto, alinhamento estruturado (estilo Vade Mecum), grifos coloridos, glosas e contadores de incidência. Feito de concurseiro para concurseiro para salvar sua visão e turbinar a revisão! Por @mau_concurseiro.
